# SAFE-GPINN PTIN Project Page

This repository hosts the public project page for:

**Actionability-Shielded Fleet Dispatch for Microgrid Formation in Weak-Communication Power-Transportation-Information Networks**

The page is a static GitHub Pages site. It includes a bilingual EN/中文 switch,
concise model and algorithm summaries, selected manuscript tables, ablation
evidence, robustness evidence and a software replay GIF generated from the
representative closed-loop trace.

Evidence files used by the page are stored under `assets/data`.

Boundary: this is a reconstructed PTIN simulation study with balanced AC checks,
SUMO/TraCI traffic evidence and ns-3 packet replay or step feedback. It is not
a hardware deployment claim.
